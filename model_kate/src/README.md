# model_kate - MVP IA Local

Estrutura baseada em Clean Architecture para o módulo/modelo de IA local do projeto Kate.

## Pastas principais:
- Presentation: Interface com o usuário (UI)
- Application: Casos de uso e orquestração
- Domain: Regras de negócio e contratos
- Infrastructure: Integrações externas (modelos LLM, arquivos, etc)

## Como evoluir
1. Implemente as interfaces e casos de uso conforme a necessidade do MVP.
2. Siga as regras do projeto (ver .copilot-rules.md).
3. Consulte os READMEs de cada pasta para orientação.
